apitext

An API that has an appetite for TEI-XML Transcriptions

what is apitext?

Apitext is a prototype RESTful Application Programing Interface (API) for TEI-XML Transcriptions. Once installed on your website, it It exposes a series of Uniform Resource Identifiers (URI). By using these identifier endpoints, a user can retrieve specific information from a TEI-XML file. Currently, our experimental student built prototype can only manage one TEI-XML file at a time and only supports URI requests from the same domain as the API itself. Future iterations of the prototype intend to address these functionality challenges.

It's probably easier to show you what our API does, rather than tell you about it. Check out a live demonstration of apitext here.

how do I install apitext?

  1. You need to have a website capable of running PHP scripts.
  2. In order for apitext to work, you will need to designate a folder on your website that will hold your TEI-XML file.
  3. You will need to install apitext into that TEI-XML folder.
    • If you are familiar with Github, you can clone the apitext repository here.
    • You can also download a zipped version of the repository here.
      • Please note, if you choose this option, you will need to open the folder inside the newly unzipped folder and change that folder name from [apitext_api-master] to [apitext_api].
      • You will then need to move that folder into your TEI-XML folder.
    • When finished your TEI-XML directory should look like this: [illustration]
  4. You will then need to move the [.htaccess] file from the [apitext_api] folder to [your TEI-XML] directory.
    5. You will then need to edit the [config.json] file located inside the [apitext_api] folder. In the [teiFolder] entry put the name of your TEI-XML folder. In the [teiFile] entry put the name of your TEI-XML file. If your folder was named [my-tei-folder] and your file was named [my-tei-xml-file.xml], then your [config.json] file would look like this: [illustration]
  5. That's it, the API should now be controlling your TEI-XML folder. Use a web browser to navigate to your file folder, you should see a welcome message.
